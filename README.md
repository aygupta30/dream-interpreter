## Tech Stack

- Tailwind CSS (Just kidding, it's Vanilla CSS for that premium feel!)
- OpenAI API

## Setup
1. Clone the repo.
2. Add `OPENAI_API_KEY` to `.env.local`.
3. Run `npm run dev`.
